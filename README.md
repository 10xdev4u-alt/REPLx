# REPLngne

**REPLngne** is a high-performance, polyglot project dedicated to building custom Read-Eval-Print Loop (REPL) engines from scratch.

## 🚀 Mission
To design, architect, and implement robust interactive shells in four distinct low-level languages, exploring the nuances of performance, memory management, and developer experience in each.

## 🛠 Implementations

This project consists of four parallel implementations, each targeting specific performance characteristics and architectural patterns:

1.  **[Rust](./rust/)**: Focus on memory safety, zero-cost abstractions, and concurrent evaluation.
2.  **[Go](./go/)**: Focus on simplicity, fast compilation, and goroutine-based concurrency.
3.  **[C](./c/)**: The "bare metal" approach. Manual memory management, raw system calls, and POSIX compliance.
4.  **[Zig](./zig/)**: Focus on modern low-level programming, comptime execution, and manual-but-safe memory layout.
